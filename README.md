# 2048 Game

## Installation
- Download Release Folder
- Open 2048.exe

## Usage
### Control
- Moving - **LeftArrow, RightArrow, UpArrow, DownArrow**

### General
- Exit - **Esc**
- Restart - **R**
- Pause - **P** or **Space**
- Undo moving - **U**
- Enable/Disable bot - **A**
- Increase/Decrease bot speed - **+**, **-**
